AttendX is a full attendance platform that replaces manual roll-calls with live face recognition — built end to end, from the database schema to three role-based dashboards.

The problem

Manual attendance wastes class time and is easy to fake with proxies. Institutions needed something fast, accurate and verifiable.

How I built it

A Flask + MySQL backend manages sessions and records. OpenCV and the face_recognition library match students against enrolled face encodings in real time, while a location check confirms the student is actually present. Admins, faculty and students each get their own dashboard, and reports export to PDF and Excel.

Outcome

A class is marked in seconds instead of minutes, attendance becomes tamper-resistant, and faculty get attendance analytics at a glance. The screenshots show the admin dashboard, the login flow, and class management.

The problem

Online shoppers can't tell how an outfit will actually look on them, which drives hesitation and returns.

How I built it

A React 19 + Vite + Tailwind single-page UI posts a person image and a garment to a Flask API, which calls a hosted diffusion try-on model (IDM-VTON) on Replicate and returns the result. An earlier prototype used a simple Pillow overlay; the current version uses the diffusion model for realistic try-on.

What's next

Refining result quality and the upload experience before opening a public demo.
